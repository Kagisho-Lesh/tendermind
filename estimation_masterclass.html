📖 How to Use This Tool — Complete Guide
1Resource Library — Browse Your Pricing Database

The Resource Library is your master pricing register — exactly like CCS Candy's Resource List. Every labour rate, plant hire rate, material price, and composite (built-up) rate lives here.

Use the Type buttons (Labour / Plant / Material / Composite) to filter by resource category. Toggle Simplex for direct rates or Complex for built-up composite rates. Use the search bar to find any resource by code or description.

Click any resource card to open a detailed modal showing the full rate build-up — just like a CCS Candy Complex Worksheet. Hover and drag any card into the Calculation Pad to use it in a BOQ estimate.

Simplex = a single resource with a direct rate (e.g., L02004 Bricklayer at R67.66/hr). Complex = a built-up composite of multiple resources (e.g., WC003 Reinforced Concrete Foundation at R4,490/m³).

2Production Rates — Visual Catalogue of Plant & Skilled Activities

The Production Rates Catalogue shows you real output rates for every type of plant and every skilled trade. This is the data behind every duration estimate you'll build.

Switch between the Plant/Equipment tab (excavators, ADTs, dozers, rollers, concrete pumps, cranes, etc.) and the Skilled Activities tab (bricklaying, shuttering, welding, steel fixing, scaffolding, painting, electrical).

Click any card to see the production formula, step-by-step worked example, application rules, and the plant code that feeds the rate. Use this to understand how production rates are derived — not just memorised.

3Calculation Pad — BOQ Builder with Drag & Drop

The BOQ Builder tab is your estimation workspace. The left panel is your Resource Palette — all resources available to drag. The right panel is your live BOQ workspace.

To build a rate: Find a resource in the palette, drag it across and drop it into the workspace. A row appears with the rate pre-filled. Enter your quantity. The amount calculates instantly.

Rows are grouped by type (Labour, Plant, Material, Composite) and the footer shows a running grand total. Use + Add Row to manually type any item. The Export button generates a formatted text summary you can copy.

The Production Calculator tab lets you select an operation, input parameters, and apply all the WBHO adjustment factors (BFF, Efficiency, Mech. Availability, Operator Skill, Grade, Rolling Resistance) to get an adjusted production rate with step-by-step workings shown.

4Variable Scriptpad — CCS Candy [Local Variables] System

The Variable Scriptpad is the most powerful feature — a live calculation scripting environment modelled on CCS Candy's local variable system.

Define any variable in [square brackets] and assign it a value or expression. All subsequent lines can reference that variable by name. Results cascade automatically — change one value and everything recalculates.

Syntax rules:

[VarName] = 2.69 → Define a variable with a number

[EffBucket] = [Bucket] * [BFF] → Reference other variables in expressions

// This is a comment → Comments are ignored in calculation

## Section Header → Section dividers for readability

Use the pre-loaded templates (Excavator Calculation, Concrete Pour, Brickwork Estimate) to see how variables chain together across a full production rate calculation — like a Python script but for construction estimating.

5Factors Library — All Adjustment Factors at a Glance

The Factors Library is your reference for every adjustment factor needed to convert theoretical production rates into realistic field rates. Never guess — look it up here.

Includes: Bucket Fill Factor (BFF) by material, Swell & Shrinkage factors, Efficiency by operator/site condition, Grade resistance, Rolling resistance by road condition, Operator skill levels, and Mechanical Availability targets by machine type.

Master formula: Realistic Rate = Theoretical Rate × BFF × Efficiency × MA × Operator × Grade × Rolling

6Programme Reference — Sasol Infill Well Activity Schedule

The Programme Reference shows the full activity schedule for the Sasol Infill Well Y project — a real-world WBHO tender programme with 117 working days and 30+ activities.

Activities are colour-coded by float: Red = Critical (0 float), Yellow = Near-critical (1–5 days), Green = Non-critical (16+ days). Filter by critical path, behind schedule, or view all activities.

Use this to understand how a real construction programme is structured — with WBS sections, dependencies, milestones, and float management.
